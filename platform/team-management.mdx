---
title: "Team Management"
description: "Invite teammates and manage users on your Mount Thor account."
---

Account administrators can invite teammates to Mount Thor. Invited users
receive an email with a sign-up link and can log in with `mthr login` after
accepting.

## Invite a Teammate

Send an invitation to a new team member:

```bash
mthr users invite user@example.com
```

Optionally assign a role and display name at invite time:

```bash
mthr users invite user@example.com \
  --role mthr-cluster-admin \
  --display-name "Jane Doe"
```

| Flag | Required | Description |
|---|---|---|
| `--role` | no | Role to assign on acceptance (e.g. `mthr-cluster-admin`, `mthr-cluster-readonly`). When omitted, the invitee receives no role until an admin assigns one. |
| `--display-name` | no | Pre-filled display name for the new user. |

The invitee receives an email with a sign-up link. After completing sign-up,
they can authenticate with `mthr login`.

<Note>
  To list team members programmatically, see the
  [Customer Users API reference](/api-reference/customer-users/get-v1admincustomer-users).
</Note>
